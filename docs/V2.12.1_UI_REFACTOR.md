## 6. 热重载问题排查记录

CDP visual smoke 首次运行失败，经多轮调试定位三个 blocker：

1. **PLUGIN_ID 错误**：脚本硬编码 `"obsidian-llm-cli-bridge"`，但 manifest.json 的 id 是 `"llm-cli-bridge"`。导致 disablePlugin/enablePlugin 操作了不存在的插件 ID，插件从未被重新加载。
2. **API 调用方式错误**：使用 `app.disablePlugin(id)` / `app.enablePlugin(id)`，但正确 API 是 `app.plugins.disablePlugin(id)` / `app.plugins.enablePlugin(id)`。前者返回 undefined（不是函数），后者才是 Obsidian 插件管理 API。
3. **layoutCheckJS 变量作用域**：`return { tab, ... }` 中的 `tab` 是外部函数参数，在生成的 IIFE 内部未定义，导致 "tab is not defined" runtime 错误。

修复方式：正确 PLUGIN_ID + `app.plugins.*` API + `tab: ${JSON.stringify(tab)}` 字符串注入 + detach view leaves 后重新打开。
