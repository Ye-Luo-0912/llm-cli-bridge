- Baseline：当前 `master`，不开新分支。
- **不 release / 不 tag**。
- **不恢复 Prompt Snippets**。

- 附件不会污染后续轮次：默认 message-scoped，发送后清空；显式 Pin 才跨轮。

## 3. RuntimeTranscript / RunStateAggregator

### 目标
不再直接 `WorkflowEvent[] -> TimelineNode[]`，引入 `RuntimeTranscript`/`RunStateAggregator` 中间层：

- `text_delta` 只追加到 final answer buffer。
- `thinking_delta` 追加到**同一个** thinking block（保证只有一个 thinking block）。
- `thinking_tokens` 只更新 thinking 标题（token 计数），不产生新 node。
- `tool_start`/`tool_progress`/`tool_result` 合并同一个 tool node（按 callId）。
- result success 只标记 completed，**不生成重复 final message**。
- `session_started` / runtime file tools / raw log 默认只进 Developer mode。

## 4. Agent UI

- `process` / `thought` / `tools`
- `final answer`（始终在最下面，流式完整展示）

- 运行中 process 展开，完成后 process 自动折叠。

- 不展示 raw event log / Session started / runtime file tools 等开发噪音（普通用户）。
