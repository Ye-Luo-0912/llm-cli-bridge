# V2.14.0-E1 - Strong Confirm External Read Approval Patch Report

## ChangedBehavior

`approvePendingExternalReadRequest()` now distinguishes normal approval from explicit strong confirmation.

- `grantRootSafety=allow`: normal approval creates a session read grant.
- `grantRootSafety=confirm`: normal approval does not create a grant.
- `grantRootSafety=confirm`: `strongConfirm=true` or `allowConfirmRoot=true` creates a session read grant.
- `grantRootSafety=deny`: approval never creates a grant, even with strong confirmation.

## StrongConfirmFlow

The pending external read UI now passes `strongConfirm=true` only from strong-confirm buttons.

- Normal allow-root UI paths pass `strongConfirm=false`.
- Strong-confirm directory approval creates a directory-scope session grant.
- Strong-confirm file approval with `forceFileScope=true` creates a file-scope session grant.
- No file content is read and no external directory is scanned.

## DeniedRootBehavior

Denied grant roots remain hard denied.

- `grantRootSafety=deny` still hides normal allow buttons in the UI.
- Direct policy approval with `strongConfirm=true` still returns the unchanged grant store.
- Deny action still only removes the pending request and never creates a grant.

## Tests

Added coverage verifies:

- Confirm root normal approval does not create a grant.
- Confirm root strong-confirm approval creates a directory grant.
- Confirm root strong-confirm plus `forceFileScope` creates a file grant.
- Deny root does not create a grant even with strong confirmation.
- Allow root normal approval still creates a grant.
- UI confirm buttons pass the explicit strong-confirm flag.
- `promptPackage`, AgentEvent v0.1, Skills runtime, CLI backend, and SDK backend remain unwired.

Required validation:

- `npm run build`
- `npm run test:unit`
- `npm run test:process`

## Recommendation

Freeze E1 after validation. The next phase can add a narrow runtime caller that creates pending read requests with `pathKind` and `knownProjectRootMarkers`, without reading file contents or injecting prompt text.
