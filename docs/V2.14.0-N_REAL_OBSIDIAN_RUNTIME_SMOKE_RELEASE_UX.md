Do not expand the self-hosted file runtime. Keep the plugin as the context, Working Set, attachment, permission boundary, and handoff bridge.

Before release, fix the local Claude Code certificate issue and rerun the native read/edit smoke against a single unambiguous test Vault. The current bridge/runtime boundary is suitable to freeze, with native execution marked as environment-limited rather than passed.
